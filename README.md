Introduction

Healthcare data is large and complex, making manual analysis difficult and time-consuming.
This project uses synthetic data and EDA techniques to identify patterns and high-risk patients.


Dataset Overview

The dataset contains 10,000 synthetic patient records with attributes like age, BP, sugar, and cholesterol.
It is generated using Python to simulate real-world healthcare data without privacy issues.


Statistical Analysis

Statistical measures like mean, median, and standard deviation are used to understand the data.
These metrics help in identifying trends and variations in health parameters.


Methodology

The project involves data generation, preprocessing, EDA, risk classification, and visualization.
Different charts and analysis techniques are used to extract meaningful insights.

Conclusion

The project successfully identifies patterns and high-risk patients using synthetic data.
It demonstrates how data analysis can support better healthcare decision-making.
